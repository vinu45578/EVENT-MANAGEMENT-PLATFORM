## Stakeholder Register

The stakeholder register provides a structured list of stakeholders along with their roles, interests, and level of influence in the **Event Management Platform**.

| Stakeholder | Role | Interest | Influence |
|------------|------|----------|-----------|
| Students | End User | View events, register, and participate | Medium |
| Student Volunteers | End User | Enroll for volunteering and track participation | Medium |
| Event Organizers | Administrator | Create and manage events efficiently | High |
| College Administration | Administrator | Monitor events and student participation | High |
| Project Manager | Project Team | Project planning and coordination | High |
| Lead Developer | Project Team | System design and implementation | High |
| QA Lead | Project Team | Quality assurance and testing | Medium |
| Documentation Lead | Project Team | Project documentation | Medium |
| Faculty / Evaluator | Academic Reviewer | Project evaluation and assessment | High |
| System Administrator | Technical Support | System maintenance and data management | Medium |

This stakeholder register helps in understanding the involvement, responsibilities, and influence of each stakeholder in the successful development and operation of the Event Management Platform.
