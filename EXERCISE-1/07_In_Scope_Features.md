## In-Scope Features

This document lists the features that are included within the scope of the **Event Management Platform** for the current project phase.

### In-Scope Features

- User registration and login (students and administrators)
- Role-based access control (Student, Event Organizer, Administrator)
- Event creation and management by administrators
- Display of upcoming and ongoing events
- Event registration for students
- Volunteer enrollment with role details
- Event schedule and venue information
- Attendance tracking for participants and volunteers
- Automated notifications and event updates
- Centralized event information dashboard
- Student participation and volunteering history
- Record maintenance for certificates and recognition
- Basic reports on event participation and attendance

These features define the functional boundaries of the system and will be implemented as part of the Event Management Platform in this project phase.
