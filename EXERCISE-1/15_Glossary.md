## Glossary

This glossary defines key terms used in the **Event Management Platform** to ensure clarity and a common understanding among all stakeholders.

| Term | Description |
|------|-------------|
| Event | An organized activity such as a workshop, seminar, fest, or volunteering program |
| Student | A user who views events, registers, and participates or volunteers |
| Administrator | A user responsible for creating and managing events |
| Event Organizer | Person coordinating and executing events within the platform |
| Volunteer | A student who assists in event activities |
| Registration | Process by which a student signs up for an event |
| Attendance | Record of participant or volunteer presence at an event |
| Participation Record | Log of events attended or volunteered by a student |
| Notification | System-generated message about event updates |
| Dashboard | Main interface displaying events and participation details |
| SRS | Software Requirements Specification |
| UML | Unified Modeling Language |
