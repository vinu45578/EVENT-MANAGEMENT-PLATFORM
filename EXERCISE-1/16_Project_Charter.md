## Project Charter

### Project Title
**Event Management Platform**

### Project Objective
The objective of the Event Management Platform is to provide a centralized and efficient system for managing college events and volunteering activities. The platform aims to improve communication, increase student participation, streamline volunteer coordination, and maintain accurate participation records for recognition and certification.

### Project Scope
The project includes features such as event creation and management, student event registration, volunteer enrollment, attendance tracking, participation history maintenance, and basic reporting for administrators and faculty.

- Internet connectivity is available during system usage.
